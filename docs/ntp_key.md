NTP Keys configuration

## 1. Ungrouped

### 1.1 key_id

**Type**: _integer_

Specifies a key_id which is used for NTP authentication.

### 1.2 trust_enable

**Type**: _boolean_

Enables trust settings for this key_id. Default is false.

### 1.3 key_password

**Type**: _string_

Specifies a key_password which is used for NTP authentication.
